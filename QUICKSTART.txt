Установка:
1. Распаковать архив в папку
2. Открыть терминал в папке проекта
3. Выполнить команду: php -S localhost:8000
4. По желанию сделать сайт публичным через CloudPub

Открыть сайт:
http://localhost:8000

ЕСЛИ НЕ РАБОТАЕТ:

- Проверьте версию PHP: php -v (нужна 7.4+)
- Порт 8000 занят? Используйте другой: php -S localhost:8080
- Ошибка БД? Удалите папку database/ и перезапустите

Основные файлы:
- index.php - точка входа, роутинг
- config/config.php - настройки
- config/database.php - БД и таблицы
- includes/functions.php - вспомогательные функции
- pages/ - все страницы сайта
